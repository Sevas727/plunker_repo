## Overview

Draw to Save Dog is a casual mobile puzzle game published on Google Play by Piggy Games. Players draw lines and shapes on the screen to protect a dog character from various hazards and obstacles across progressively challenging levels.

## Key Features

- **Draw Mechanics**: Freehand drawing physics-based puzzles
- **Progressive Difficulty**: Increasingly complex hazard scenarios
- **Casual Gameplay**: Short sessions suitable for mobile gaming
- **Ad Monetization**: Free-to-play with ad-supported model

## Tech Stack

- **Engine**: Unity
- **Language**: C#
- **Architecture**: Zenject (DI), Scriptable Objects, State Machines
- **Animation**: Spine (2D skeletal), DOTween
- **Backend**: Firebase (Analytics, Crashlytics, Firestore)
- **Monetization**: Unity Ads SDK, Google AdMob
- **Platform**: Android (Google Play)
