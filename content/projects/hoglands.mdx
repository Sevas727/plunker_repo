## Overview

Hoglands is a dynamic strategy game with roleplaying elements set in a gloomy medieval world with hand-drawn graphics. Players defend their kingdom from undead hordes while uncovering the source of a spreading plague. Published on Steam by Piggy Games.

## Key Features

- **Day/Night Combat Cycles**: Gather resources and hunt vampires by day, defend against stronger enemy waves by night
- **Army Building**: Recruit survivors, construct defenses, and launch attacks
- **Resource Management**: Collect materials, build infrastructure, manage supplies
- **Dark Mechanics**: Sacrifice subjects to evil gods for combat bonuses
- **Hand-Drawn Art**: Unique visual style with detailed 2D animations

## Tech Stack

- **Engine**: Unity
- **Language**: C#
- **Architecture**: Zenject (Dependency Injection), State Machines, Scriptable Objects
- **Animation**: Spine (2D skeletal animation), DOTween
- **Lighting**: Unity 2D lighting system (Directional, Point, Spot)
- **Networking**: Firebase (Firestore for data, Analytics)
- **Asset Management**: Addressables for optimized loading
- **Platforms**: Windows (Steam)
