## Overview

Pigs Wars: Plague Kingdom is an action strategy mobile game published on Google Play by Piggy Games. Set in a medieval world, players command pig armies to defend their kingdom against waves of enemies in a mix of tower defense and RTS gameplay.

## Key Features

- **Army Management**: Recruit and upgrade units with different combat roles
- **Tower Defense**: Build and position defensive structures
- **Wave-Based Combat**: Progressively harder enemy waves with boss encounters
- **Resource Economy**: Gather and manage resources for army expansion
- **Hand-Drawn Art**: Stylized 2D medieval visuals

## Tech Stack

- **Engine**: Unity
- **Language**: C#
- **Architecture**: Zenject (DI), Scriptable Objects, State Machines
- **Animation**: Spine (2D skeletal), DOTween
- **Lighting**: Unity 2D lighting (Directional, Point, Spot)
- **Backend**: Firebase (Analytics, Crashlytics, Firestore)
- **Asset Management**: Addressables
- **Platform**: Android (Google Play)
