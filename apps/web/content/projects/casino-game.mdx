## Overview

A fully functional browser-based casino slot machine game, built from scratch in **1 day**. The game features smooth spine animations powered by PixiJS, immersive sound effects via Howler.js, and a responsive layout that works across devices.

## Key Features

- **Slot Machine Mechanics**: Spinning reels with randomized symbol outcomes
- **Spine Animations**: Smooth character and UI animations using pixi-spine
- **Sound Effects**: Ambient sounds and win/spin audio via Howler.js
- **State Management**: Redux Toolkit for predictable game state
- **Responsive Design**: Adapts to different screen sizes with viewport scaling

## Tech Stack

- **Framework**: React 19 + TypeScript
- **Rendering**: PixiJS 8 for WebGL-powered 2D graphics
- **Animations**: pixi-spine for skeletal animations
- **Audio**: Howler.js for cross-browser sound playback
- **State**: Redux Toolkit for centralized state management
- **Build**: Vite for fast development and optimized production builds
